# Sat1 ESPHome Components

Vendored Satellite1 hardware components used by `satellite1-TaterTimer.yaml`.

Source: `FutureProofHomes/Satellite1-ESPHome`, `develop` commit `e137048`.

These files are kept in-tree so the Tater Sat1 firmware does not depend on the FutureProofHomes ESPHome component repo at build time. The upstream ESPHome component license is included in `LICENSE`.
