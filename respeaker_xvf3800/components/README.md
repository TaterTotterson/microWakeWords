# ReSpeaker XVF3800 ESPHome Components

Vendored ReSpeaker XVF3800 hardware components used by respeakerXVF3800-TaterTimer.yaml.

Source: TaterTotterson/Respeaker-XVF3800-ESPHome-integration, main commit dce21b1.

These files are kept in-tree so the Tater ReSpeaker XVF3800 firmware does not depend on the separate XVF3800 ESPHome component repo at build time.
